# Como o Skip Funciona

À medida que você escreve seu código-fonte em Swift e SwiftUI, o plugin Skip para Xcode cria continuamente o código equivalente para Android. Desenvolva bibliotecas multiplataforma ou aplicativos inteiros, com desempenho nativo e interfaces de usuário nativas em ambas as plataformas.


## Vantagens do Skip

<div class="feature-grid">

  <FeatureCard title="SwiftUI, do Início ao Fim" icon="widgets" color="blue" link="/docs/modules/skip-ui/#supported-swiftui">
    Escreva SwiftUI real uma vez e obtenha uma interface de usuário nativa de primeira classe em ambas as plataformas.
    O Skip traduz as visualizações do SwiftUI para o Jetpack Compose, preservando o layout,
    a acessibilidade e as convenções da plataforma.
  </FeatureCard>

  <FeatureCard title="Sem Runtime. Sem Inchaço." icon="bolt" color="amber" link="/docs/modes/">
    O Skip faz seu trabalho em tempo de compilação. Você obtém código nativo do início ao fim,
    sem um tempo de execução gerenciado ou um motor pesado.
    Seu aplicativo Android depende apenas de bibliotecas de código aberto padrão.
  </FeatureCard>

  <FeatureCard title="Desempenho Nativo por Design" icon="rocket_launch" color="red" link="/docs/project-types/">
    Como os aplicativos Skip usam código nativo real, seus aplicativos têm uso de memória
    previsível, tempos de inicialização rápidos e características de desempenho idênticas
    ao Swift e Kotlin escritos à mão.
  </FeatureCard>

  <FeatureCard title="IDEs de Primeira Linha" icon="code" color="purple" link="/docs/app-development/">
    Continue usando o Xcode como seu principal ambiente de desenvolvimento Swift.
    Construa, depure e refatore exatamente como sempre fez.
    Use o Android Studio para personalização específica da plataforma.
  </FeatureCard>

  <FeatureCard title="Interoperabilidade Perfeita com Android" icon="sync_alt" color="green" link="/docs/platformcustomization/">
    Precisa de um SDK ou biblioteca específica do Android?
    O Skip interopera de forma limpa com o Kotlin e o Jetpack Compose, para que você possa
    integrar APIs do Android sem lutar contra o framework.
  </FeatureCard>

  <FeatureCard title="Código Aberto, Construído para o Longo Prazo" icon="lock_open" color="teal" link="/blog/skip-is-free/">
    O Skip e seus frameworks principais são gratuitos e de código aberto.
    Sem aprisionamento tecnológico, sem tempos de execução proprietários e sem esperar que os fornecedores
    acompanhem os últimos lançamentos de plataformas.
  </FeatureCard>
