A [Cursor Agent Skill](https://github.com/asserusama/skip-tool-skill) is available to help with Skip development inside AI-powered editors. The skill guides getting started (new or existing project, `skip checkup`), [mode selection](/docs/modes/) (Skip Fuse vs Skip Lite), and points to references for the CLI, compiler directives, building, troubleshooting, and porting.

Install it via [skills.sh](https://skills.sh) (`npx skills add https://github.com/asserusama/skip-tool-skill --skill skip-tool`), the [Claude Code plugin](https://github.com/asserusama/skip-tool-skill#option-b-claude-code-plugin), or by cloning into `~/.cursor/skills/skip-tool`. See the [skip-tool-skill repository](https://github.com/asserusama/skip-tool-skill) for details and usage.
